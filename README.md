# dotfiles

![Arch Linux](https://img.shields.io/badge/Arch%20Linux-1793D1?style=flat&logo=arch-linux&logoColor=white)

Personal dotfiles for Arch Linux + Sway.

![Desktop Screenshot](/pictures/desktop.png)

## Stack

- **OS**: Arch Linux
- **WM**: Sway
- **Shell**: Zsh
- **Terminal**: Ghostty
- **Editor**: Neovim
- **Bar**: Waybar

## Setup

```bash
# Stow command to create system links
make all

# Stow delete to remove all system links
make delete 
```
